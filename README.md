# webhook-leyu-notes

本仓库用于归档和发布多个独立 HTML 页面。这些页面内容彼此独立，不针对任何单一域名或具体网站。

## 项目简介

该项目是一个 HTML 页面的归档集合。页面通过仓库的静态托管功能对外提供访问，适合作为一些临时或长期需要公开访问的页面的存放地。

仓库本身不包含后端逻辑，所有页面均为纯前端 HTML 文件。

## 目录说明

- `pages/` — 存放所有独立 HTML 页面源文件
- `assets/` — 页面引用的样式、脚本、图片等资源文件
- `docs/` — 额外文档或说明文件（可选）

每个 HTML 文件代表一个独立页面，文件名即页面标识。

## 页面归档说明

页面内容由仓库维护者定期或按需更新。归档的页面类型包括但不限于：

- 信息展示页
- 工具或功能页面
- 临时通知或公告页
- 其他需要公开访问的静态页面

每个页面均为自包含的 HTML 文件，不依赖外部服务（除 CDN 资源外）。

## 使用方式

如需访问归档页面，可通过 GitHub Pages 或其他静态托管服务将本仓库部署后，直接访问对应 HTML 文件的路径。

例如，若仓库部署在 `https://example.com/`，则 `pages/example.html` 可通过 `https://example.com/pages/example.html` 访问。

## 维护说明

- 新增页面：将 HTML 文件放入 `pages/` 目录，并确保引用资源路径正确。
- 更新页面：直接修改对应 HTML 文件。
- 删除页面：移除文件即可，无需额外操作。
- 资源管理：公共资源建议放入 `assets/` 目录，避免重复引用。

仓库维护者应确保页面内容不包含敏感信息或违反平台规定的链接。

## 许可

本仓库仅用于归档和发布 HTML 页面，不涉及具体许可协议。页面内容版权归各自作者所有。
